# Restaurant-chatbot-api

Restaurant Chatbot API is a .NET Core Web API for managing restaurant reservations across channels like websites, apps, and chatbots. It handles booking requests, checks availability, and prevents double bookings using a scalable, clean architecture, making it ideal for automated reservation systems.

## How to run this project in local

1. Clone this project to your local

```bash
git clone "url"
```

2. Pull for latest changes frequently from master branch

```bash
git pull origin master
```

3. Build the dotnet app

```bash
dotnet build
```

4. Run the dotnet app

```bash
dotnet run
```
